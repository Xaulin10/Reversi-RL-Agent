# Reversi (Othello) com Agente de Aprendizagem por Reforço

Este projeto apresenta o desenvolvimento de um agente inteligente adaptativo para o jogo Reversi (Othello), utilizando técnicas de Aprendizagem por Reforço. O agente aprende a jogar a partir do zero através da interação com o ambiente do jogo, sendo posteriormente integrado num jogo totalmente funcional com interface gráfica.

O objetivo principal do projeto é aplicar, de forma prática, os conceitos de Inteligência Artificial e Aprendizagem por Reforço abordados em contexto académico, incluindo modelação de agentes, treino, avaliação e análise de desempenho.

---

## Funcionalidades Principais

- Jogo Reversi (Othello) totalmente funcional com interface gráfica
- Agente inteligente baseado em Aprendizagem por Reforço
- Treino autónomo do agente a partir do zero
- Avaliação do desempenho contra diferentes adversários
- Visualização de estatísticas e curva de aprendizagem
- Estrutura modular e organizada do código

---

## Tecnologias Utilizadas

- **Python 3**
- **Pygame** – Interface gráfica
- **Matplotlib** – Visualização de resultados
- **JSON** – Armazenamento dos pesos aprendidos
- **CSV** – Armazenamento de estatísticas de desempenho

---

## Requisitos de Software

- Python 3.10 ou superior
- Sistema operativo: Linux, Windows ou macOS

---

## Instalação das Dependências

As bibliotecas necessárias podem ser instaladas utilizando o `pip`:

```bash
pip install pygame matplotlib
```
## Execução do Jogo

Para executar o jogo com o agente inteligente:

```bash
python3 reversi.py
```
